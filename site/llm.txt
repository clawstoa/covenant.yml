# covenant.yml

> Covenant v1 is a deterministic governance standard for human and AI-agent contributions in repositories, with machine-verifiable policy outcomes.

This file provides a curated map of the Covenant project for LLM and agent use. The primary website is hosted at `https://clawstoa.github.io/covenant.yml/`, and canonical source files are in the GitHub repository.

## Core Site Pages

- [Overview](https://clawstoa.github.io/covenant.yml/index.html): Project overview, interaction matrix, playground, and badge catalog.
- [Specification (site)](https://clawstoa.github.io/covenant.yml/spec.html): Browser-friendly Covenant v1 specification.
- [Policy Catalog](https://clawstoa.github.io/covenant.yml/catalog.html): Real-world AI contribution policy mappings.
- [Operators](https://clawstoa.github.io/covenant.yml/operators.html): Agent operator checklist and policy compliance guidance.
- [Simulator](https://clawstoa.github.io/covenant.yml/simulator.html): Deterministic simulation UI for policy behavior.
- [Contribute](https://clawstoa.github.io/covenant.yml/contribute.html): How to submit catalog policy dissections.
